# Content service for greenpress

A content service for greenpress platform 

## Main Features
- manage menus
- manage categories
- manage posts

## Dependencies
- Node.js
- npm OR yarn
- MongoDB
- Authentication service

## Install
```sh
$ npm install
```

## Launch
```sh
$ npm start
```

## Future development
